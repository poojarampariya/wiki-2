WM | IT SEC & ISO 27001 | All : VPN connection issue

Created by Khushali Solanki, last modified on Mar 24, 2021

Case 1: Error while establishing VPN connection

  • Go to https://vpn.windmill.ch and login

  • Download again the profile and delete this profile on openVPN app

  • If you are not able to login, then go to https://security.google.com and try and check if your username/password is working.

Case 2: If you are facing the below error.

  • Please upgrade the OpneVPN Application latest version by login here https://vpn.windmill.ch

Case 3: “Authentication Failed” or In case you are unable to connect after changing Google Account Password

  • You can check the last password change here https://security.google.com

  • Go into the VPN application > click edit and update the new password & authenticate via code.

Case 4: “Google Authenticator code is incorrect” - Even after Multiple trials

  • Uninstall the Google Authenticator and install again and tried to connect to VPN

For any other issue Kindly contact System Administrator Saurabh Rohit
