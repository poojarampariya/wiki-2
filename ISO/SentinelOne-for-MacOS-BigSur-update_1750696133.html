Created by Khushali Solanki, last modified on Jan 11, 2021

Instructions

Note : Follow below instructions using admin account on device.

Contact CISO Khushali Solanki or Anastasiia Didenko (Deactivated) to get EmployeePassphrase and Site token.

  • Open your terminal by following this path -  Finder > Applications > Utilities > Terminal.  

  • Enter your terminal and type: sudo sentinelctl unprotect --passphrase "EmployeePassphrase".

  • Contact CISO to get your sentinel passphrase

If you get a message advising that username is not in the sudoers file, this means you are not logged in as admin: please enter command, "su Windmill" (where "Windmill" is admin login) and then repeat   sudo sentinelctl unprotect command. You may need to enter the admin password once or twice during this process.  

  • Then please type this command: sudo sentinelctl uninstall --local

  • Restart your computer

  • Update your Mac OS

  • Download the sentinel package from given link below and enter the token: Download the sentinel package from given link below and enter the site token you get from CISO:
