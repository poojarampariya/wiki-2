WM | IT SEC & ISO 27001 | All : Generate Bitbucket Recovery code

Created by Khushali Solanki on May 20, 2021

Recovery codes work exactly the same as regular verification codes for 2step verification while logging to Bitbucket.

If you don't have your mobile device or security key, you can access your account using a recovery code. When you first set up two-step verification, we gave you recovery codes and told you put them somewhere safe in case you need them.

Steps to generate recovery code :

  • Go to https://bitbucket.org/dashboard/overview

  • Click on Your profile icon → Personal settings

  • Click Two-step verification under the security tab

  • Click Show Recovery code → Click on Save option → Download

  • Create a folder in “My Drive” with the name “Bitbucket_RecoveryCodes“ and Save these downloaded recovery codes file in My drive/Bitbucket_RecoveryCodes
