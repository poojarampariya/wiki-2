Created by Khushali Solanki, last modified on May 31, 2021

Windmill uses the SentinelOne Platform for endpoint security protection and monitoring .

According to Gartner Research, An endpoint protection platform (EPP) is a solution deployed on endpoint (computer, server or mobile) devices to prevent file-based malware attacks, detect malicious activity, and provide the investigation and remediation capabilities needed to respond to dynamic security incidents and alerts.

The software is installed on each PC to monitor the health of the system.

About SentinelOne :

SentinelOne is an Endpoint Protection Platform which is mean to prevent, detect, respond, and hunt threats to the security of Windmill computer assets. SentinelOne proactively monitors incoming and outgoing traffic on computers it‚Äôs installed on, and protects against cyber attacks then contains and ultimately resolve threats before they can become damaging to WMSS computers, platforms or data.

The goal of the SentinelOne platform is to eliminate exposure to internal and external cyber threats.

To learn more about SentinelOne, please visit: https://www.sentinelone.com/platform/

Instructions

The Software installations will be assisted by WM Security team. Please change the hostname on your computer and contact CISO/DevOps for the installation support.

Device Hostname change

In order to properly identify your protection status, please change your current hostname, associated with your laptop in the following way: WMSS-Location-OS-FirstName

For instance:¬† WMSS-IN-WIN10-Khushali

Here the location is India - So IN¬†

OS is Windows so WIN10 (Mac users should put WMSS-Location-Mac-FirstName)

The instructions on how you can do it on your OS quickly are described below

For Windows OS

  1. In the search box on the taskbar, type control panel, and then select Control Panel

  2. Click on System and Security > System

  3. ¬†Under Section ‚ÄúComputer name, domain, and work group settings‚Äù Click on Change Settings

  4. It will open the System Properties dialog box. Click on Change as given below.

Type the new name for your computer in the following way: WMSS-Location-OS-FirstName and click OK

5. Restart your computer, so the hostname will be automatically updated after that

For Mac OS device

  1. Open System Preferences

  2. Click on Sharing

  3. Click on the field Computer Name: at the top

  4. Highlight the current name and enter a new one

  5. Press enter or click anywhere in the window outside of the text field to set the new name
